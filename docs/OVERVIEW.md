# Overview

## About LactaLink

LactaLink is a tech-driven platform designed to facilitate the donation and distribution of breast milk. Built with modern technologies like **React Native (Expo)** and **Next.js**, it connects donors, recipients, hospitals, and milk banks to ensure safe and efficient breast milk sharing.

## Purpose and Goals

The primary purpose of LactaLink is to create a reliable and secure ecosystem for breast milk donation and distribution. The platform aims to:

- Bridge the gap between donors and recipients.
- Ensure medical approval and safety in the donation process.
- Provide convenience for individuals and organizations involved in breast milk sharing.

### Outcomes Achieved

- Streamlined communication between donors, recipients, hospitals, and milk banks.
- Enhanced accessibility to breast milk for those in need.
- A user-friendly interface for managing donations and requests.

## Key Features

- **Mobile App (React Native)**: A seamless experience for donors and recipients on Android and iOS.
- **Admin Panel (Next.js)**: A web-based interface for managing users, identity verifications, donations, requests, etc.
- **GIS Integration**: Location-based services powered by Google Maps API for finding nearby donors, recipients, hospitals, and milk banks.
- **Authentication**: Secure login with Supabase Auth and Google OAuth.
- **Identity Verification**: Optional ID verification using face-api.js for enhanced trust and security.
- **Realtime Messaging**: Instant communication between users using Supabase Realtime.
- **Email Notifications**: Automated email services for updates and communication.
- **Scalable Infrastructure**: Hosted on Vercel and integrated with Supabase for database management.

## Relevance

LactaLink addresses a critical need for safe and efficient breast milk sharing, empowering communities and healthcare organizations to support infant nutrition. By leveraging modern technology, it ensures accessibility, safety, and convenience for all stakeholders.
